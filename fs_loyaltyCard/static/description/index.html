Loyalty Card Module

Odoo 14.0 Community

Author: SARL FOCUS SYSTEM - contact@focussystem.dz

Description

Give customers a discount for each Sale order due to their member card

Card category

You can manage card category at the menu Sales -> Configuration -> Loyalty -> Card category

Card Period

You can manage card period at the menu Sales -> Configuration -> Loyalty -> Card period
Have 2 types: Month and Year

Card type

You can manage card type at the menu Sales -> Configuration -> Loyalty -> Card type

Loyalty Card

You can manage the loyalty card at the menu Sales -> Configuration -> Loyalty -> Card
