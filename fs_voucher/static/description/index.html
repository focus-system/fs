Odoo 14.0 Community

Author: SARL FOCUS SYSTEM - contact@focussystem.dz

Description

The creation and use of vouchers.

Vouchers

When selling a voucher: Check the box: is a voucher, then enter its number.
When the invoice related to this sale is paid, a payment for this customer is generated. In this case, a payment refers to a voucher.
This payment will be used for the next purchases of this customer.
